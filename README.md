## Rodando o projeto

No terminal, clone o projeto:
<br>
`git clone https://github.com/luzialeticia/api-reprograma-blog.git`

Entre na pasta do projeto:
<br>
`cd api-reprograma-blog`

Instale as dependências:
<br>
`yarn install`

Execute:
<br>
`yarn start`
